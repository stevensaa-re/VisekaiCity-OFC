📜 Server Rules

Harap patuhi semua peraturan berikut agar server tetap nyaman untuk semua pemain:

  • 1. Dilarang Griefing (merusak bangunan orang lain)
  • 2. Dilarang menggunakan Cheat/Hack
  • 3. Dilarang exploit bug untuk keuntungan pribadi
  • 4. PvP hanya di area yang sudah ditentukan
  • 5. Jangan menghancurkan rumah atau properti pemain lain
  • 6. SURVIVAL ONLY! Jangan minta creative/items
  • 7. Hormati sesama pemain, no toxic behavior
  • 8. Saling membantu dan supportif
  • 9. Server aktif Jumat—Minggu & hari libur
  • 10. Jam operasional: 05.00 - 20.00 WIB

⚠️ Pelanggaran rules akan mengakibatkan:

• Warning pertama

• Kick dari server

• Ban permanent (untuk pelanggaran berat)

💬 Group Rules

Peraturan untuk Discord dan WhatsApp Group VisekaiCity:

  • 1. Toxic sewajarnya, jangan berlebihan
  • 2. Dilarang membahas konten 18+
  • 3. Boleh bahas cewek, asal tetap sopan dan respect
  • 4. Boleh off-topic, tapi jangan spam berlebihan
  • 5. Hindari promosi tanpa izin admin
  • 6. Hormati semua member dan admin
  • 7. Jaga privasi dan data pribadi orang lain

💡 Tips untuk pengalaman terbaik:

• Be friendly dan ramah ke semua orang

• Aktif ikut diskusi dan event

• Help newbie yang baru join

• Report masalah ke admin jika ada
